# Python Utilities

This folder contains special utilities written in SAS to help making the usage of the SAS Viya APIs easier.

Find a short description and example usage of each below.

## [Access SAS Viya APIs](./accesssas__viya_apis.py)

Retrieves the SAS session bearer token and the URL of the SAS Viya server. Next a request is made against the files endpoint and the result is returned as a SAS data set.
